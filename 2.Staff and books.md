## Pay attention to the following:
- Calculate the `iss_date` and `return_date` values.
- You may either write a date as a string in the following format `YYYY-MM-DD` or use the `date()` function to calculate a date.  
  For example, `date('now')` will return the current date. Check out this [link](https://www.sqlite.org/lang_datefunc.html) for more details.
- For `return_indicator` with delivery information: `0 = False`, `1 = True`.
